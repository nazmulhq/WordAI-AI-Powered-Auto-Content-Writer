== Installation ==
1. Install WordAI Auto Content Writer by uploading the plugin to the '/wp-content/plugins/' directory inside.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. First put your openAI [API key](https://platform.openai.com/account/api-keys) through 'API Settings' page of the plugin.
4. Then you are ready to generate auto content / images for your Wordpress!

It will help you to write fast content/images automatically based on your input word/words within a minute.
